# Over Permission Detector

## 📌 Overview

Over Permission Detector is a simple web-based tool designed to analyze and detect unnecessary or excessive permissions in applications. It helps improve security awareness by highlighting potential risks.

---

## 🚀 Features

* Detects over-permission usage
* Simple and user-friendly interface
* Dashboard to view results
* Lightweight and fast

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

### Method 1: Open Directly

1. Extract the ZIP file
2. Open the project folder
3. Double-click `index.html`

---

### Method 2: Run with Local Server (Recommended)

#### Using VS Code:

1. Open project in VS Code
2. Install **Live Server extension**
3. Right-click `index.html` → **Open with Live Server**

#### Using Python:

Run the following command in the project folder:

```
python -m http.server
```

Then open:

```
http://localhost:8000
```
